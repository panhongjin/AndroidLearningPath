# 操作系统引论

## 1.1 操作系统的目标和作用

### 1.1.1 操作系统的目标

目前存在着多种类型的OS，不同类型的OS，其目标各有所侧重。通常在计算机硬件上配置的OS，其目标有以下几点：
- 方便性
- 有效性
- 可扩充性
- 开放性


### 1.1.2 操作系统的作用

1. 从计算机系统组成观点----C操作系统是系统软件
计算机系统组成：
软件：应用软件<br>
      系统软件(操作系统、语言处理系统和常用的例行服务程序)<br>
硬件：输入/输出(I/O)设备<br>
      存储器(内存)<br>
	  中央处理器(CPU)
2. 从资源管理程序观点----操作系统是系统资源管理者
- 操作系统是系统资源管理程序，用于控制和管理计算机系统的硬件和软件资源。<br>
